    3. Longest Substring Without Repeating Characters

    Given a string, find the length of the longest substring without repeating characters.

    Solution

    Sliding window. Use sliding window. For instance: ‘pwwkewp’, answer is 4 ‘kewp’

    1. check if the character inside the dictionary
    2. if yes, update begin pointer to the next index (we don’t need to count the repeating char into res)
    3. update res, current length is current - begin + 1, if only 1 char, would be 1.
    4. update dictionary.

    204. Count Primes

    Count the number of prime numbers less than a non-negative number, n.

    Solution

    Sieve of Eratosthenes

    In order to avoid time exceed exception, tag the non-primes numbers based on the known primes. Build a list to store the prime state.

    For example, n = 20. we only need to check range(2, n**0.5), because we will tag non-prime from prime square. 5 * 5 is 25 greater than 20.

    Therefore, nums = list(range(2, int(n ** 0.5) + 1)).

    When we tag 2, multiply start from 2, till 9, [4, 6, 8, 10, 12, 14, 16, 18].

    When we tag 3, multiply start from 3, till 6, skip all number that already tagged, [9, 15]

    When we tag 4, 4 already in non-primes list. skip it.

    Count the amount of True in prime list.
