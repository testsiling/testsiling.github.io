Solution 2: Digital Root

Explanation: N = a0 * 10^0 + a1 * 10^1 + a2 * 10^2 + … an * 10^n Example: 234 = 4 * 10^0 + 3 * 10^1 + 2 * 10^3

1 % 9 = 1, 10 % 9 = 1,
100 % 9 = 1, …

a1 % 9 + a2 % 9 + … + an % 9 = (a1 + a2 + … + an) % 9

(a * b) % n = (a % n) * (b % n)

Hence: N % 9 = a0 % 9 + a1 % 9 + a2 % 9 + … + an % 9 Because: a[i] < 10 So: a[i] % 9 = a[i] Thus: N % 9 = a0 + a1 + a2 + … + an
