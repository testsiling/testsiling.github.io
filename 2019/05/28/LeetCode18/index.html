75. Sort Colors

The idea is pretty straightforward. Set up 2 pointers: p0 as the right boundary of 0, and the p2 as the left boundary of 2.

Also needs a pointer cur as the current color.

if cur is 0, swap nums[cur] and nums[p0], because now cur is 0, cur++, also, p0++

if cur is 2, swao nums[cur] and nums[p2], p2 -=1. doesn’t need to move cur because the number that swaped to cur didn’t compare with 0 and 2.
