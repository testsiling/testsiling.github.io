    Double Linked List

    Implement a double linked list, just like using a ordered dictionary (ordered dictionary is implemented by double linked list, reference)

    Use a dictionary to log key and the representative node. Everytime put into a pair of (k,v), remove the original node, add it to tail. Then check capacity, if exceed, remove the first node.

    Everytime get a value, remove the original node, add it to tail. get valu from node.val
