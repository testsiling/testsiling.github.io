    42. Trapping Rain Water

    Dynamic Programming

    Calculate all left highest wall and right highest wall for each point. Then calculate the water.

    Two Pointers

    If left higher than right, then right responsible to how many water this point get, update right pointer. If reverse, then left responsible to the water volume, update left pointer.

    121. Best Time to Buy and Sell Stock

    prices[i] - prices[i-1]] is today’s difference, add the profit the day before, get the difference between buy day and sell day.

    if less than 0, today is a low price, buy in this day, profit is 0 today.
