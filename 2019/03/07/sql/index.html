    DDL (Data Definition Language)

    CREATE

    Create an EMPLOYEE table as follows:

    • Set the primary key constraint as SSN, and the foreign key as SUPER_SSN. Do not yet specify the foreign key constraint referencing the department table.

    • Use constraint name for PK (EMPPK) and FK (EMPSUPERFK) constraint. (Naming the constraint is helpful in disabling/enabling later.)

    • DISABLE all the constraints for now. (NOT NULL does not have to be disabled).

    Create a DEPARTMENT table as follows:

    • Create DEPARTMENT table with PK as DNUMBER and FK as MGR_SSN

    • Use constraint name for PK (DEPTPK) and FK (DEPTMGRFK)

    • DISABLE constraints.

    ALTER TABLE

    • ALTER TABLE to enable PK constraint of DEPARTMENT table.
    • ALTER TABLE to add foreign key specification of DNO in the EMPLOYEE to the DEPARTMENT table and name the constraint (EMPDEPTFK).
    • Can check all constraints
