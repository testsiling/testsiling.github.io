    127. Word Ladder

    1. build a dictionary for all generic states, Ex. *og, h*t …
    2. from the begin word, push all possible words for all generic states into the queue
    3. use visited dictionary
    4. if go through all posible way for that generic states still no answer, mark as [] in dictionary, means bad end.
