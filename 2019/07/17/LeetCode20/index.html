997. Squares of a Sorted Array

Time Complexity is O(n). For example, list [-4,-1,0,3,10]

The greatest number must come from square of (-4) and (10), tures out that 10**2 = 10 is the greatest.

The second greatest number must come from square of (-4) and (3), tures out that (-4)**2 = 16 is the second greatest.

Use 2 pointers to log the leftest and the rightest elements.

76. Minimum Window Substring

This problem is almost my first problem in Leetcode. Idea is pretty easy, but hard to find an elegant way to solve it.

  • missing is the amount to missing char

  • set 2 pointer2 left and right, both start from index 0.

  • when current right element is target, missing -= 1.

  • found all target when missing == 0. Then eveluate the window size. update it. remember missing

  • pop out the first found element (update missing and required). Then find next substring that meets requirement.

  • when met requirement again, update left pointer to the leftest valid element (skip chars that don’t need or already met the required amount.)

986. Interval List Intersections

find the max intersection. Find which interval end first, A or B, jump to next

left = max(A[0], B[0])

right = min(A[1], B[1])
