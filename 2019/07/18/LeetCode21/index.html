990. Satisfiability of Equality Equations

This one is my first Union find problem.

Union all equal expression first, then go through unequal exp.

Update element’s ancestor every time.

For example, [“f==a”,“a==b”,“f!=e”,“a==c”,“b==e”,“c==f”]

Initial: {a: a, b: b, c: c, e: e, f: f }

  1. “f==a” : {a: a, b: b, c: c, e: e, f: a}
  2. “a==b” : {a: b, b: b, c: c, e: e, f: a}
  3. “a==c” : {a: b, b: c, c: c, e: e, f: a}
  4. “b==e” : {a: b, b: c, c: e, e: e, f: a}
  5. “c==f” : {a: e, b: c, c: e, e: e, f: e}

547. Friend Circles

Union-find

First is general union find. Then update all union to root. count all unit root.

Example:

first found: [1, 2, 2]

Then update to root: [2, 2, 2]

So return 1.
