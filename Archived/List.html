## Array

  • fixed length
  • All boxes hold the same type
  • get one reference when its created
  • if you reassign the reference, you can never get the array back.

Three valid notation: - y = new int[3];//32 * 3 = 96 bit - x = new int[]{1,2,3}; - int[] z = {4,5.6}; //declaring a variable

Two ways to copy arrays 1. 循环操作每个item 2. arraycopy

Method arraycopy

Fast but hard to read

Five param: 1) Source array 2) Start position in source 3) Target array 4) Start position in target 5) Number to copy

Result: x = {4,5,6,1,2,9}

2D Array

Exercise: x[0][0]=? w[0][0]=?

Answer: x[0][0]=-1 w[0][0]=1 z[0][0] = -z[0][0]; 这句，z[0][0] 与 x[0][0] 指向同一个field, 所以 x[0][0] 的 item变了。
